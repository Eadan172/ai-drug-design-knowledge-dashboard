功能特性
7大板块：
1、化学小分子	虚拟筛选、分子生成、化学信息学、量子化学、分子动力学、性质预测
2、多肽	肽链折叠预测、环肽设计、序列分析、药物设计、抗菌肽设计
3、RNA	RNA结构预测、mRNA设计、RNA药物设计
4、抗体	抗体人源化、抗原-抗体对接、SingleB平台、抗体药物设计、MHC-肽预测
5、PROTAC	PROTAC设计、Linker设计
6、天然产物	基因组挖掘、结构预测、活性预测
7、虚拟细胞	单细胞分析、药物响应预测、通路建模

文献综述模块
展示10篇精选核心文献
支持按日期和影响因子(IF)排序
每篇文献展示标题、作者、期刊、年份、引用数、摘要
提供查看全文链接（PubMed/DOI/arXiv）
最新高分文献模块
按技术点分类（虚拟筛选、分子生成、分子动力学、性质预测等）
每个技术点整理5篇文献
优先展示有代码的文章，提供代码链接
显示期刊、年份、引用数

技术学科知识模块
经典教材（Amazon等购买链接）
在线课程（Coursera、edX、Stanford等免费/付费课程）
技术博客（开源教程、官方文档）

研发公司模块（需要就职小伙伴看过来！提供部分公司参考）
国际公司：DeepMind、Insilico Medicine、Atomwise、Recursion、Schrödinger等
国内公司：晶泰科技、英矽智能、未知君、星药科技、燧坤智能、分子之心、深势科技等
